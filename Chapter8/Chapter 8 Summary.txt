1. 트리란 무엇인가?
>> 부모노드와 자식노드로 이루어진 자료구조

2. 이진트리의 표현 방법엔 무엇이 있는가?
>> 1) 배열 : 노드 번호를 인덱스 삼아 저장한다. 기억공간의 낭비가 있다.
   2) 연결리스트 : 가장 일반적인 트리 표현 방식이다. 연산이 조금 복잡하다.

3. 이진트리의 순회는 어떻게 이뤄지는가?
>> 1) 전위순회 : 현재 노드의 데이터를 처리하고 왼쪽, 오른쪽 서브트리로 도는 방식이다. 
   2) 중위순회 : 왼쪽 서브트리의 데이터들을 처리하고, 현 노드의 데이터 처리 후, 오른쪽 서브트리의 데이터를 처리한다.
   3) 후위순회 : 왼쪽, 오른쪽 서브트리들의 데이터를 처리하고 현 노드의 데이터를 처리하는 방식이다.

4. 이진트리의 연산엔 무엇이 있는가?
>> insert() : 노드를 삽입한다.
   isEmpty() : 트리가 비었는지 검사한다.
