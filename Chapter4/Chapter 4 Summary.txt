1. 큐란 무엇인가?
>> 선입선출(FIFO)의 구조를 가진 자료구조이다.

2. 큐의 추상 자료형은?
>> 객체 : 선입선출의 접근 방법을 유지하는 요소들의 모음
>> 연산 : enqueue() : 후단을 통해 요소를 삽입
          dequeue() : 전단을 통해 요소를 삭제
          isFull() : front와 rear의 차이가 1이면 true
          isEmpty() :front와 rear가 똑같다면 true
          peek() : 전단의 요소 반환
	  size() : 큐의 모든 요소들의 갯수 반환

3. 덱이란 무엇인가?
>> 전단과 후단 모두를 통해 접근할 수 있는 자료구조이다.
